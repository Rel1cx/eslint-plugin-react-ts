# debug/function-component

## Rule category

Debug.

## What it does

Reports all function components. Useful for debugging. This rule should only be used for debugging purposes. Otherwise, leave it off.

## Examples

```tsx twoslash
import React from "react";

function Example() {
  return <button />;
}
```

```tsx twoslash
import React from "react";

function Example() {
  return React.createElement("button");
}
```

```tsx twoslash
import React from "react";

const Example = () => <button />;
```

```tsx twoslash
import React from "react";

const Example = () => React.createElement("button");
```

```tsx twoslash
import React from "react";

const Example = React.memo(() => <button />);
```

```tsx twoslash
import React from "react";

const Example = React.forwardRef(() => <button />);
```
