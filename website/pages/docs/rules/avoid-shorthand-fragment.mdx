# prefer-shorthand-fragment

## Rule category

Style.

## What it does

Enforces the usage of fragment shorthand syntax.

## Examples

### Failing

```tsx twoslash
import React from "react";

function Example() {
  return (
    <>
      <button />
      <button />
    </>
  );
}
```

### Passing

```tsx twoslash
import React, { Fragment } from "react";

function Example() {
  return (
    <Fragment>
      <button />
      <button />
    </Fragment>
  );
}
```
