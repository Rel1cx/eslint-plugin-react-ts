# no-string-refs

## Rule category

Restriction.

## What it does

Disallows using deprecated `string refs`.

## Why is this bad?

String refs are deprecated in React. Use callback refs instead.

## Examples

### Failing

```tsx twoslash
import React from "react";

function Example() {
// @error: Do not use string refs
  return <div ref="ref" />;
}
```

### Passing

```tsx twoslash
import React, { useRef } from "react";

function Example() {
// @annotate: Use 'useRef' refs instead
  const ref = useRef<HTMLDivElement>(null);
  return <div ref={ref} />;
}
```
