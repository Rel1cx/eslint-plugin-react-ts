# Function: narrow()

> **narrow**\<`TType`\>(`a`): [`Narrow`](../type-aliases/Narrow.md)\<`TType`\>

Infers embedded primitive type of any type
Same as `as const` but without setting the object as readonly and without needing the user to use it.
